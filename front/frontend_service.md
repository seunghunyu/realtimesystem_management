# realtimesystem_management
실시간 감지 및 분석 시스템 개발 샘플<br>

<h2>1.기획의도</h2>
  SW아카데미 교육 멘토링 마지막 멘토링에 학생들의 이해를 좀 더 돕기 위해 샘플 프로젝트를 진행하게 되었음.
<br><br>
<h2>2.구성 </h2>
 -Front : TypeScript + React  <br>
 -Backend : SprintBoot  <br>
<br><br>
<h2>3.사용 오픈 소스</h2>
    -Kafka<br>
    -Redis<br>
    -mariadb<br>
    -hadoop(환경이 되면.. 없으면 mariadb로 대체, 어차피 데이터가 많이 없을거라) <br>
    -Google Tagmanager(학생들은 Google Tagmanager가 아닌 샘플 태깅 js를 따로 만들어서 시킬 생각, 멘토링 기간에 비해 학습비용이 너무 많이 듬)<br>
<br><br>
<h2>4.프로젝트 서버 구성</h2>
    -쇼핑몰 Front 서버(TypeScript + React) <br>
    -관리자 Front 서버(TypeScript + React) <br>
    -쇼핑몰 Backend 서버(Springboot)<br>
    -관리자 Backend 서버(Springboot)<br>
